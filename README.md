# PathAI Engine — Studor DS Screening Project

Three analytical systems built on the Open University Learning Analytics Dataset (OULAD):

- **Task 1 — Behavioral Scoring:** Per-student per-week engagement score (0–100) and three-archetype K-Means clustering.
- **Task 2 — Disengagement Prediction:** Week-6 XGBoost binary classifier with calibrated probabilities, threshold tuning, SHAP-driven advisor alerts.
- **Task 3 — Course Recommendations:** Top-3 course suggestions via content-based and collaborative-filtering recommenders, with a three-tier cold-start fallback.

Full methodology and results are in `outputs/PathAI_Combined_Report.pdf`.

---

## Project Structure

```
task1_behavioral_scoring.py   # Feature engineering → 0-100 weekly engagement score
task1_archetypes.py           # K-Means archetype clustering + trajectory visualisation
disengagement_features.py     # Week-6 feature matrix (Task 1 outputs + demographics)
disengagement_model.py        # Train / calibrate / evaluate XGBoost classifier
disengagement_alert.py        # Generate example advisor alert payloads (TP / FP / borderline)
task3_reco_features.py        # Student and course profile vectors for recommendation
task3_content_based.py        # Content-based recommender (cosine similarity + Wilson prior)
task3_collab_filter.py        # Item-kNN and user-kNN collaborative filtering
task3_cold_start.py           # Three-tier cold-start dispatcher
task3_evaluate.py             # Temporal holdout evaluation + alpha grid search
requirements.txt
outputs/                      # All generated artefacts (see below)
```

Key outputs (all generated by the scripts above):

## Dataset

Download the **OULAD** dataset from [Kaggle](https://www.kaggle.com/datasets/anlgrbz/student-demographics-online-education-dataoulad) and place all 7 CSV files in the project root:

```
assessments.csv  courses.csv  studentAssessment.csv  studentInfo.csv
studentRegistration.csv  studentVle.csv  vle.csv
```

`studentVle.csv` is ~460 MB and is streamed in chunks — do not attempt to load it fully into memory.

---

## Setup

```bash
python -m venv .venv
source .venv/bin/activate      # Windows: .venv\Scripts\activate
pip install -r requirements.txt
```

Python 3.10+, ~4 GB RAM peak, ~500 MB disk for outputs.

---

## Running

Run scripts in this order — each step depends on the outputs of the previous:

```bash
# Task 1
python task1_behavioral_scoring.py    # ~3-5 min — produces weekly_scores.csv
python task1_archetypes.py            # ~30 sec — produces archetypes.png

# Task 2
python disengagement_features.py      # builds week6_features.csv
python disengagement_model.py         # trains, calibrates, evaluates
python disengagement_alert.py         # generates example alert payloads

# Task 3
python task3_reco_features.py         # builds student + course profile vectors
python task3_content_based.py
python task3_collab_filter.py
python task3_cold_start.py
python task3_evaluate.py              # temporal holdout evaluation + alpha tuning
```

All outputs are written to `outputs/` (created automatically). The pipeline is fully deterministic — fixed random seeds throughout.
